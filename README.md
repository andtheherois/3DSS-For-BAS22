# 3DSS-For-BAS22

This is the 3D Shader scopes module for Boomsticks and Sharpsticks 22 which requires the GAMMA's BAS22 files. This will not work with BAS22 of MODDB. Requires Sniper, Rifle, LMG, and/or SMG Modules
